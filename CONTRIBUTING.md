# contributing to allahdroid

спасибо, что хочешь помочь проекту! это прошивка для эмулятора, основанная на AOSP 5.1.1, с юмором и чёрным юмором. прежде чем что-то делать, прочитай это.

## что за проект

allahdroid - юмористическая прошивка с элементами сатиры. Никакого отношения к религии, только технические шутки.

## как помочь

### 1. сообщить о баге
создай [Issue](https://github.com/pristochelovek097/allahdroid-rom/issues), опиши проблему и как её воспроизвести.

### 2. предложить фичу
тоже через Issue, с меткой `enhancement`.

### 3. написать код
- форкни репозиторий.
- создай ветку: `git checkout -b feature/что-то-крутое`
- сделай изменения.
- запушь и открой Pull Request.

### 4. улучшить документацию
исправить опечатки, добавить примеры, уточнить инструкции.

## стиль кода

- отступы: 4 пробела
- имена переменных: `snake_case`
- комментарии: на русском или английском
- не добавляй лишних зависимостей без обсуждения

## сборка

см. [README.md](README.md) — там есть инструкция по сборке и запуску.
